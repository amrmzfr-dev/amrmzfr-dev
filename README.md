  ### 🚀 Production Projects

  **Ozone Machine IoT System**
  ![ESP32](https://img.shields.io/badge/ESP32-0082C9?style=for-the-badge&logo=arduino&logoColor=white&height=25)
  ![Go](https://img.shields.io/badge/Go_/_Gin-00ADD8?style=for-the-badge&logo=go&logoColor=white&height=25)

  - Full-stack IoT system tracking ozone machine usage across client sites in production.
  - ESP32 firmware with NTP sync, NVS-backed clock fallback, and offline retry queue for reliable session delivery.
  - Go/Gin REST API with API key auth and UTC+8 analytics; PostgreSQL for persistence.
  - Per-client React + TypeScript dashboard with billing summary, device health, and session log.

  **CCTV API Server** *(Perodua Q VE EV Charger Programme)*
  ![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white&height=25)
  ![AWS](https://img.shields.io/badge/AWS_EC2-232F3E?style=for-the-badge&logo=amazonaws&logoColor=white&height=25)

  - REST bridge over Dahua's C NetSDK (ctypes bindings), integrated into the P Circle consumer app by an external agency.
  - Three-layer auth: admin key, client API keys, and per-user stream tokens with expiry.
  - Deployed on AWS EC2 with Nginx TLS, Gunicorn, and RDS MySQL — live in production.
